## Design Features

### Color Scheme
- **Header Gradient**: `from-sky-600 to-sky-700` with white text
- **Card Background**: `bg-gray-50` with `border-gray-200`
- **Primary Buttons**: `bg-sky-600 hover:bg-sky-700`
- **Info Boxes**: `bg-blue-50 border-blue-200`
- **Success Indicators**: Green accents
- **Warning Indicators**: Yellow/Amber accents

### Typography
- **Header**: `text-xl font-bold` (white on gradient)
- **Section Headers**: `text-lg font-semibold` with icons
- **Labels**: `text-sm font-medium text-gray-700`
- **Help Text**: `text-xs text-gray-500`
- **Body**: `text-sm text-gray-900`

### Spacing
- **Modal Padding**: `p-6`
- **Card Padding**: `p-4`
- **Section Spacing**: `space-y-6`
- **Input Spacing**: `mb-4` or `mb-2`

### Interactive Elements
- **Dropdowns**: Full width with border, focus ring
- **Input Fields**: Rounded borders, focus states
- **Checkboxes**: Sky color scheme
- **Buttons**: Rounded corners, hover states, disabled states
- **Icons**: Lucide React icons, contextual colors

### Responsive Design
- **Max Width**: `max-w-2xl` for single mode, `max-w-4xl` for bulk mode
- **Mobile**: Stack vertically, full width inputs
- **Tablet**: Maintain grid layout
- **Desktop**: Optimal spacing and layout

### Accessibility
- **Focus Indicators**: Visible focus rings
- **ARIA Labels**: All interactive elements
- **Keyboard Navigation**: Tab order, Enter to submit, Escape to close
- **Color Contrast**: WCAG AA compliant

### Visual Hierarchy
1. **Header** - Gradient background draws attention
2. **Cards** - Clear separation of sections
3. **Icons** - Visual cues for each section
4. **Buttons** - Clear call-to-action
5. **Info Boxes** - Secondary information
